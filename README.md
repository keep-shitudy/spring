# 참여방법
1. 이름 혹은 github ID로 브랜치를 생성한다.
2. 생성된 브랜치에 자신의 필기를 업로드 한다. (폴더명에 콜론":" 안됩니다!)
3. 이때 commit 규칙을 지키도록한다!
4. 원본 저장소로 Pull Request를 한다. 
5. 주의!! main은 건들지 않는다!! 개인 폴더 내부만 수정해주세요 :)
